# music

## 项目运行

### 下载依赖

```
npm install
```

### 运行项目
```
npm run serve
```

### 项目打包
```
npm run build
```

## 项目简介

基于Vue.js开发的一款移动端音乐播放器app，数据通过调用**[网易云音乐接口](https://neteasecloudmusicapi.vercel.app/#/)获取。**

## 实现功能

1. **播放列表：** 显示当前播放列表，查看播放模式，**清空列表**，点击收藏，点击删除，点击添加歌曲到当前播放列表
2. **添加歌曲到播放列表：** 搜索歌曲添加，播放历史添加，搜索历史添加
3. **播放组件：** 拖动进度条跳转播放进度，歌词跟随进度联动
